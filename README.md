## Prerequisite:
- Node (v16.0 or later)
- npm (v8.0 or later)
- A MongoDB Atlas account

## How to compile:
1. Create a MongoDB Auth String,
2. Create a "apiKyes.js" file in src folder, and input the text below:
`exports.apiKeys = {
  MONGO_AUTH:
    "[REPLACE THIS WITH YOUR AUTH STRING]",
};`

1. npm install
2. nodemon .
